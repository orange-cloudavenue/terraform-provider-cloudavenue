```release-note:feature
`resource/cloudavenue_backup` - New resource to manage NetBackup feature. The `cloudavenue_backup` resource allows you to manage backup strategy for `vdc`, `vapp` and `vm` from NetBackup solution. [Please refer to the documentation for more information.](https://wiki.cloudavenue.orange-business.com/wiki/Backup)
```

```release-note:feature
`datasource/cloudavenue_backup` - New datasource to manage NetBackup feature. The `cloudavenue_backup` data source allows you to retrieve information about a backup of NetBackup solution.
```
