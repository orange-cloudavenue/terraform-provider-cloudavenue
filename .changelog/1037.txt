```release-note:new-resource
`resource/cloudavenue_edgegateway_services` - This resource enables users to manage Edge Gateway network services in CloudAvenue. It serves as the foundation for configuring network services in your Edge Gateway within the CloudAvenue environment.
```

```release-note:new-data-source
`datasource/cloudavenue_edgegateway_services` - This data source allows users to retrieve information about Edge Gateway network services in CloudAvenue. It provides a way to query and obtain details about the available network services associated with your Edge Gateway.
```
